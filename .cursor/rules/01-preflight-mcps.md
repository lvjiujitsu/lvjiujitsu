# Preflight and MCP Rule

Antes de qualquer tarefa relevante:
- validar `.cursor/` e `.cursor/rules/`;
- validar alinhamento documental;
- validar internet quando necessária;
- validar cada MCP relevante na sessão atual;
- registrar modo degradado se algum falhar.

Se houver MCP relevante para a tarefa, ele deve ser usado.
